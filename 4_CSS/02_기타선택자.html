1.속성 선택자

선택하고자 하는 요소에서 작성되어있는 속성을 이용해서 선택하는 방법
선택자 뒤에 []를 이용해서 속성과 속성값을 제시하면서 선택한다 

        선택자[속성=속성값] 
        선택자[속성~=속성값] 
        선택자[속성|=속성값]
        선택자[속성^=속성값]
        선택자[속성$=속성값]
        선택자[속성*=속성값] {
            스타일속성 : 값;
        }

2. 자손선택자와 후손 선택자

요소들이 중첩되어 작성 가능
자손 : 바로 하위인 요소들 / 후손 : 하위요소 전부 

        * 자손선택자 : >
        a>b{
            스타일속성 : 값;
        }

        *후손선택자 : 공백
        a d{
            스타일속성 : 값;
        }

3. 동위(같은레벨) 선택자

동위관계(같은레벨)에 있는 뒤에 위치한 특정 요소들 선택할 때 사용한다.

        * a요소 바로뒤에 있는 b요소 "하나만"을 선택할때
        a+b{
            스타일속성 : 값;
        }

        * a요소 뒤에 "모든" b요소 전부 선택
        a~b{
            스타일속성 : 값;
        }

4. 반응선택자

사용자의 움직임에 따라 선택되는 선택자

*해당 요소에 클릭이 되었을 경우 스타일을 부여한다. 
선택지:active{
    스타일속성:값;
}

* 해당 요소에 마우스가 올라가는 순간 스타일 부여한다 
선택자:hover{
    스타일속성:값;
}

5. 상태선택자

요소의 상태에 따라서 선택되는 선택자

        1) 체크된(checked)상태의 요소
        선택자:checked{
            스타일속성:값;
        }
    
        2) 초정(focus)이 맞춰진 input 요소 선택
        선택자:focus{
            스타일속성:값;
        }
아이디 :
비밀번호 : 
        3)활성(enabled), 비활성(disabled)되어있는 요소 선택

        선택자:enabled{
            스타일속성:값;
        }
        선택자:disabled{
            스타일속성:값;
        }
